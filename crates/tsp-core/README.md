# TSP Core

This crate provides core types and utilities for solving the Traveling Salesman Problem (TSP).
It provides the foundation for other crates in [`tsp-rust`](https://github.com/RaoulLuque/tsp-rust).

[![Build status](https://github.com/RaoulLuque/tsp-rust/workflows/ci/badge.svg)](https://github.com/RaoulLuque/tsp-rust/actions)
[![](https://img.shields.io/crates/v/tsp-core.svg)](https://crates.io/crates/tsp-core)

[https://docs.rs/tsp-core](https://docs.rs/tsp-core)


### Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
tsp-core = "0.1"
```
